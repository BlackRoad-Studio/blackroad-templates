API Reference

Complete reference documentation for the BlackRoad REST API. Build powerful integrations with our platform.

Base URL
https://api.blackroad.io/v2
GET /users Auth required

List all users

Returns a paginated list of users in your organization. Results are sorted by creation date in descending order.

Query Parameters

Name Type Description
limit integer optional Number of results per page (1-100). Default: 20
cursor string optional Pagination cursor from previous response
status string optional Filter by status: active, inactive, pending
Request 
curl -X GET "https://api.blackroad.io/v2/users?limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
200 Success
Response 
{
  "data": [
    {
      "id": "usr_1234567890",
      "email": "alexa@blackroad.io",
      "name": "Alexa Mundson",
      "role": "admin",
      "status": "active",
      "created_at": "2026-01-15T10:30:00Z"
    }
  ],
  "meta": {
    "total": 156,
    "cursor": "eyJpZCI6MTU2fQ=="
  }
}
POST /users Auth required

Create a user

Creates a new user in your organization. An invitation email will be sent automatically.

Request Body

Name Type Description
email string required User's email address
name string required User's full name
role string optional User role: admin, developer, viewer. Default: developer
Request 
curl -X POST "https://api.blackroad.io/v2/users" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "new.user@company.com",
    "name": "New User",
    "role": "developer"
  }'
201 Created
Response 
{
  "id": "usr_0987654321",
  "email": "new.user@company.com",
  "name": "New User",
  "role": "developer",
  "status": "pending",
  "created_at": "2026-02-05T14:00:00Z"
}
DELETE /users/{user_id} Auth required

Delete a user

Permanently deletes a user from your organization. This action cannot be undone.

Path Parameters

Name Type Description
user_id string required The unique identifier of the user
Request 
curl -X DELETE "https://api.blackroad.io/v2/users/usr_1234567890" \
  -H "Authorization: Bearer YOUR_API_KEY"
204 No Content
