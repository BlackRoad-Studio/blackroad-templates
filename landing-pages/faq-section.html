Frequently asked questions

Everything you need to know about BlackRoad.

What is BlackRoad?

+
BlackRoad is an operating system for autonomous AI agents. It provides the infrastructure to deploy, orchestrate, and scale thousands of intelligent agents that can learn, adapt, and work together to accomplish complex tasks. Think of it as Kubernetes for AI agents.

How do I get started?

+
Getting started is easy! Sign up for a free account, install our CLI with npm install -g @blackroad/cli, and deploy your first agent with blackroad deploy. Our quickstart guide will have you running agents in under 5 minutes.

What's included in the free tier?

+
The free tier includes up to 3 agents, 1,000 API calls per month, community support, and basic analytics. It's perfect for learning the platform and building proof-of-concepts. No credit card required to get started.

How does persistent memory work?

+
Each agent has access to an append-only memory journal that persists across sessions. Memory entries are verified using PS-SHA-∞ hashing for data integrity. Agents can query their memory, learn from past experiences, and share knowledge with other agents in your workspace.

Is my data secure?

+
Absolutely. We're SOC2 Type II certified and all data is encrypted at rest and in transit. Our PS-SHA-∞ quantum-resistant hashing ensures complete data integrity. Enterprise customers can also use their own encryption keys and deploy in private cloud environments.

What's the uptime SLA?

+
We offer 99.99% uptime SLA for Pro and Scale plans. Our infrastructure is deployed across multiple availability zones with automatic failover. Check our status page for real-time availability information.

Can I self-host BlackRoad?

+
Yes! Enterprise customers can deploy BlackRoad in their own cloud environment (AWS, GCP, Azure) or on-premises. We provide full deployment automation and ongoing support. Contact our sales team for more information.
