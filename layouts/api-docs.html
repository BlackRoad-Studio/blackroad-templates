Create Agent

Spawn a new agent with the specified configuration. The agent will be initialized with default memory and begin listening for events on its assigned channels.

POST https://api.blackroad.io/v2/agents

Request Body

Parameter Required Description
name
string
 required Unique identifier for the agent. Must be lowercase alphanumeric with hyphens.
role
string
 required The agent's role, which determines its default capabilities and event subscriptions.
config
object
 optional Additional configuration options including memory settings, timeout values, and custom metadata.
replicas
integer
 optional Number of agent instances to spawn. Defaults to 1. Max 1000 per request.

Example Request

curl -X POST https://api.blackroad.io/v2/agents \
  -H "Authorization: Bearer br_prod_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "analyst-001",
    "role": "data-processor",
    "config": {
      "memory": "persistent",
      "timeout": 30000
    },
    "replicas": 10
  }'
